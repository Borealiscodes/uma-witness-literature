# Unified Meta-Architecture (UMA), Sādhana, and MPAM

This repository contains three tightly related pieces:

- **UMA (Unified Meta-Architecture):**  
  A conceptual, multi-plane meta-architecture for thinking about systems, structure, and governance.

- **Sādhana:**  
  An ethical, non-dual stance for how we relate to systems, power, and meaning.

- **MPAM (Multi-Plane Attachment Method):**  
  A documentation method for architectures that span multiple planes (structural, governance, developmental, dynamic, integrative).

This is **conceptual and exploratory work**.  
It is meant for thinking, modeling, and learning—not for running real-world institutions or exercising authority.

## Ethical Core

This repository is grounded in three simple vows:

- **No architecture without ethics.**  
- **No governance without humility.**  
- **No map mistaken for the territory.**

The intent of this work is to explore safer, clearer ways of thinking about complex systems, not to create a new authority or doctrine.
